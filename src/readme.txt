To run this react application first download the zip file then extract all the files. After that open the project in your VsCode editor then install the dependies using "npm i".  After that run the application using "npm start" command.


Find files: 

In the component folder there is moviecard.js containing the details of the movie as a card.
In our project in our public folder we have our json data in movies.json file.
Screenshots are inside the screenshot folder inside the src folder.
CSS is in the app.css file.
